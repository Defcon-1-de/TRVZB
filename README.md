# TRVZB
Intelligente Heizungssteuerung für Home Assistant: Steuert beliebige Climate‑Thermostate per zentralem Wochen‑Schedule, Fensterkontakt, manuellem Override und Außentemperatur‑Differenz. Nutzt Snapshot/Restore des vorherigen Zustands beim Lüften und funktioniert herstellerunabhängig mit allen  climate ‑Entitäten.
